# Lifeon Portfolio - Premium Glassmorphism Experience

A high-end, interactive portfolio website featuring a modern glassmorphism design, dynamic theme switcher, and a custom music player. Built with HTML, Vanilla CSS, and JavaScript.

## ✨ Features
- **Premium Glassmorphism UI**: Stunning blurred backgrounds and smooth transitions.
- **Dynamic Theme Switcher**: Change accent colors and backgrounds on the fly.
- **Custom Music Player**: Integrated playlist with category support (VN/Foreign) and responsive controls.
- **Responsive Design**: Optimized for both high-resolution desktops and mobile devices.
- **Interactive Widgets**: Digital clock, date, and dynamic profile editing.

## 🚀 How to Run Locally
1. Clone the repository or download the source code.
2. Ensure you have the `file mp3/` directory with your music files.
3. Open `index.html` in any modern web browser.
   - *Note: For the best experience with the music player, it is recommended to run this via a local server (like Live Server or XAMPP) to ensure metadata loads correctly.*

## 📂 Project Structure
- `index.html`: The main structure and logic of the application.
- `style.css`: The comprehensive design system and responsive styles.
- `file mp3/`: Directory containing the music assets.

## 🛠 Technology Stack
- HTML5
- CSS3 (Custom Variables, Flexbox, Grid, Glassmorphism)
- JavaScript (ES6+)
- Font Awesome (Icons)
- Google Fonts (Outfit & Dancing Script)
