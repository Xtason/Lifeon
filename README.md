# Lifeon - Premium Glassmorphism Portfolio

Lifeon là một bản giao diện Portfolio hiện đại, tập trung vào trải nghiệm người dùng với phong cách thiết kế **Glassmorphism** (kính mờ) sang trọng, được tối ưu hóa cho cả máy tính và thiết bị di động.

## ✨ Tính năng nổi bật
- **PWA Ready**: Có thể cài đặt trực tiếp trên điện thoại như một ứng dụng (Ẩn thanh trình duyệt).
- **Thiết kế Glassmorphism**: Hiệu ứng kính mờ cao cấp với các micro-animations mượt mà.
- **Trình phát nhạc tích hợp**: Hỗ trợ danh sách phát, thanh tiến trình và điều khiển trực quan.
- **Chế độ chỉnh sửa trực tiếp**: Cho phép thay đổi thông tin, hình ảnh và màu sắc ngay trên trình duyệt (Lưu vào LocalStorage).
- **Responsive**: Hiển thị hoàn hảo trên mọi kích thước màn hình, bao gồm cả chế độ xoay ngang của điện thoại.

## 🚀 Hướng dẫn cài đặt lên GitHub Pages
1. Tải toàn bộ mã nguồn này về máy.
2. Tạo một Repository mới trên GitHub.
3. Upload toàn bộ các file và thư mục lên Repository đó (bao gồm cả thư mục `file mp3`).
4. Vào phần **Settings** -> **Pages** của Repository.
5. Ở mục **Build and deployment**, chọn Branch là `main` (hoặc `master`) và folder là `/ (root)`. Nhấn **Save**.
6. Chờ khoảng 1-2 phút, bạn sẽ có một link website chính thức!

## 📱 Cách cài đặt thành App trên điện thoại
- **iOS (Safari)**: Nhấn nút Share -> Chọn "Thêm vào MH chính".
- **Android (Chrome)**: Nhấn dấu 3 chấm -> Chọn "Cài đặt ứng dụng".
